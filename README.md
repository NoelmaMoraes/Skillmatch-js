# SkillMatch JS

## Sobre o projeto

O SkillMatch JS é um simulador simples de compatibilidade entre um candidato e vagas de front-end júnior.

O sistema compara as habilidades do candidato com os requisitos das vagas cadastradas e apresenta:

- percentual de compatibilidade;
- habilidades encontradas;
- habilidades faltantes;
- vaga mais compatível;
- recomendação de estudo.

O projeto foi desenvolvido utilizando JavaScript puro, com foco em lógica de programação e estruturação de código.

## Como executar

Este projeto não precisa de Node.js.

Você pode executar o projeto diretamente pelo navegador:

1. Abrir o Google Chrome;
2. Pressionar `F12` ou `Ctrl + Shift + J`;
3. Abrir a aba `Console`;
4. Copiar todo o conteúdo do arquivo `skillmatch.js`;
5. Colar no console do navegador;
6. Pressionar `Enter`.
